# Student Grade Calculator

## Project Description

A comprehensive grade calculator that processes multiple students' marks, calculates grades with personalized comments, stores results using lists, and provides class statistics. The application demonstrates the use of conditionals, loops, lists, functions, error handling, file operations, and formatted output.

## Project Objectives

* Collect marks for multiple students.
* Validate all user inputs.
* Calculate averages and grades.
* Provide personalized comments.
* Generate class statistics.
* Search for specific students.
* Save results to a file.
* Practice Python fundamentals through a complete project.

---

## What I Learned

### 1. Conditional Logic

Using `if`, `elif`, and `else` statements to assign grades and comments based on average marks.

### 2. Lists

Using lists to store student names, marks, averages, grades, and final results.

### 3. Loops

Using `for` loops to process multiple students and `while` loops for input validation.

### 4. Error Handling

Using `try-except` blocks to handle invalid inputs and prevent program crashes.

### 5. Functions

Creating reusable functions for grade calculation, statistics generation, searching, and saving results.

### 6. File Handling

Writing student results to a text file for future reference.

### 7. Data Processing

Calculating averages, highest scores, lowest scores, and class statistics.

## Grading System

| Average Score | Grade | Comment                                |
| ------------- | ----- | -------------------------------------- |
| 90 - 100      | A     | Excellent! Keep up the great work!     |
| 80 - 89       | B     | Very Good! You're doing well.          |
| 70 - 79       | C     | Good. Room for improvement.            |
| 60 - 69       | D     | Needs Improvement. Please study more.  |
| 0 - 59        | F     | Failed. Please seek help from teacher. |

## Challenges & Solutions

### Challenge: Handling Invalid Marks Input

**Solution:** Used `while` loops with `try-except` blocks to ensure only valid numeric values between 0 and 100 are accepted.

### Challenge: Formatting the Results Table

**Solution:** Used Python f-string formatting with fixed-width columns for alignment.

### Challenge: Calculating Multiple Statistics

**Solution:** Used built-in functions such as `sum()`, `max()`, `min()`, and list indexing.

### Challenge: Searching for Students

**Solution:** Implemented a case-insensitive search using loops and string comparison.
